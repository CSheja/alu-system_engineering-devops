# Firewall Configuration

This project folder contains scripts to configure firewall rules on Ubuntu servers.

## Files

- firewall/0-block_all_incoming_traffic_but: Bash script to configure UFW to block all incoming traffic except ports 22, 80, and 443.

# System Engineering and DevOps

This repository contains configurations for setting up a secure firewall using UFW on a web server. The goal is to block all incoming traffic except for SSH (22), HTTP (80), and HTTPS (443) traffic.

## Files

- `firewall/0-block_all_incoming_traffic_but`: A script that configures the UFW firewall to allow only necessary ports.
